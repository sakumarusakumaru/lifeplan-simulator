## [3.4.0] - 2026-04-29

### Added
- 右上ヘッダーに「初期化」ボタンを追加。確認ダイアログ経由でプランをデフォルト値にリセット可能。ホバー時は赤色で危険操作を示す

### Fixed
- ヘッダーロゴの「v3」→「ver3」に変更し、文字を小さく・細く・薄く表示
- 「税金・社会保険（VER2）」の「（VER2）」を削除
- 収入セクションのラベル「入社」「退職」→「入社年齢」「退社年齢」に変更
- BALANCE SHEET の「不動産投資（評価額）」ラベルを1行表示に修正（`whitespace-nowrap`）
- BALANCE SHEET の凡例に「負債」と「純資産」の間の黒い区切り線を追加
- 詳細入力ページの右エリア最上部のHealthHeaderをコンパクト1行表示に変更（スコア+4KPI横並び）

## [3.3.0] - 2026-04-27

### Added
- 新NISA対応: 投資信託・株を「NISA口座（非課税）」と「課税口座」に分割入力
- NISAは名目リターンそのまま、課税口座は実効リターン（×0.79685）で試算
- 取り崩し順序: 課税口座を先に取り崩し、NISAを最後まで温存するデフォルト設定
- AssetSnapshot / DrawAsset 型に `fNisa` / `sNisa` を追加

### Changed
- AssetsSection・AssetsMegaSection・SettingsMegaSection の DrawAsset ラベルをすべて更新

---

## [3.2.0] - 2026-04-26

### Added
- トップページにFP免責の同意ゲートを設置（5項目チェックボックス）
- `/terms` 利用規約ページ（11条）
- `/privacy` プライバシーポリシーページ（クライアントサイドのみのデータ処理を明示）
- フッターに目立つ免責ブロックと利用規約・プライバシーポリシーへのリンク

### Changed
- FP資格者に限定されるような表現を「参考情報」「一般的に論じられます」等へ中立化
- ConsentDialog: 初回訪問時にモーダルで同意確認

## [3.1.0] - 2026-04-25

### Added
- ヘッダーに JSON ダウンロード・読み込みボタン（詳細入力タブのみ）
- ヘッダーに PDF ダウンロードボタン（全タブ共通）
- `@media print` CSS で A4 印刷レイアウト最適化
- シナリオ比較ページを拡充: インフレ感度・収入増加・支出削減・積立増額・運用利回り・早期退職等8シナリオ

### Changed
- ヘッダーロゴを「LIFE PLAN SIMULATOR v3」に統一
- V2へ戻るボタンを削除

## [3.0.0] - 2026-04-22

### Added
- `/v3` ルートとして v3 を本番化
- 左カラムのセクションヘッダーをスティッキー表示
- ANNUAL CASHFLOW グラフに `stackOffset="sign"` を適用し収入バーを正しく表示
- BALANCE SHEET のハイブリッドカラム表示（流動/固定 区切り + ホバーインタラクション）
- HealthHeader に 100点満点スコア・FP相談レポート・取り崩し戦略提案を追加
- 金・コモディティ資産カテゴリを追加
- 不動産の現在評価額を物件種別×築年数で自動算定（手動上書き可）

### Changed
- v2 を安定版として `/v2` に保存し、v3 を開発版として分離
- フッターをコンパクト1行レイアウトに変更

## [2.0.0] - 2026-04-10

### Added
- 12セクションを4つのメガセクション（基本設定・収入・資産・支出）に再編
- CollapsibleSubGroup コンポーネントによる折りたたみ UI
- 税金詳細モード（VER2）: 所得税累進・住民税・社会保険・介護保険を厳密計算
- 確定拠出年金（DC）の取り崩しを60歳以降に制限
- 取り崩しカスタム順序 UI（ドラッグ不要・ボタン型）
